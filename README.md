## Background
I made this project because I was struggling with my time management especially in "Pengetahuan Kuantitatif" questions.
There we expected to solve a mathematic question with only one minute. I was frustated because after 9 months there still only like half of the question that I can solve perfectly
Worse, I can't found an alarm in the internet that can repeat the alarm until some number amount of time in some period. So, basically I don't have a good environment that simulate the pressure.
I need the pressure to make my brain work and get used to it. So, I made this alarm

## Timer Description
This alarm can produce a sound as much as the number of question that you need. 
After, a period of number that you insert to the system, this alrm willl produce a "beep" sound.
This allow you to immediately switch number after the beep sound. I suggest you to do that so you can know how short the time that you get in each question.
This will be repeated until the number of question fulfilled

## How to use it
1. Input the number of question that you want to answer in a column with "Q". Then, input the number of time that you need to answer one question
(column "Min" show the number of minutes and column "sec" show the number of second)
2. The system will make a sound after the countdown reach 00:00. It all will be repeated ultil all question is answered. 
For example: If you want to do 4 question each 10 second. Then the timer will do countdown from 10 to 0 and an alarm will be turned on. Then, the timer will do a countdown again and sounding an alarm again. For 4 times
3. To start the timer click the button "start"
4. To pause the timer click the button "Pause" and to start the timer again click the "Pause" botton once more.
5. To stop the session click the buttom "Stop" It will reset the timer and input.
6. The Total time  show the time for one session It is basically the number of Question * Time per one question
7. There will be a pop up button when the session is finished and when you enter wrong input. Continue by clicking the "Ok" button
8. Leaving the column empty made the system automatically input 0
